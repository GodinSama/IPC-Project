# IPC Project - Route Manager

Not my cup of tea, nonetheless, still a very nice project. UI could be improved as well as some functions in the map area, etc.

## Features

* **Activity Tracking:** Import GPX files to get a full breakdown of your runs, including distance, speed, average pace, and elevation gain.
* **Map View:** See your route overlaid on regional maps with color-coded start and end markers. Includes an interactive elevation profile graph.
* **Custom Annotations:** Drop points, text, lines, or circles to mark specific spots on your route.
* **User Profiles:** Fully functional registration, login, avatar support, and session history tracking.

## Tech Stack

* **Language:** Java
* **Framework:** JavaFX
* **Database:** SQLite

## Installation & Setup

Trying out the project is as easy as downloading the release and running it—no installation required!

* Download the latest `.zip` file from the **Releases** tab.
* Extract the entire folder to your computer.
* Run the `.exe` file found inside.

**Note:** Please do not edit, move, or delete any of the files or folders sitting next to the `.exe` (especially the `maps` folder or the `.dll` files).
  
  The program relies on them to run properly and will crash if they are separated!
